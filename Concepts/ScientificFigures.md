Visualization is one of the most important aspects of scientific research. Here are some resources to help navigate Adobe Illustrator and Biorender that I use in my work.  
### Adobe Illustrator  
A versatile tool, but can have a steep learning curve for beginners. Scientists can fall back on the following:  

1. Adobe Illustrator Tutorials

The official Adobe website offers a range of tutorials from beginner to advanced levels, covering basic tools to more complex techniques.
Illustrator for Biologists by Martin Krzywinski

This is a fantastic resource specifically designed for scientists. It provides detailed, step-by-step instructions on how to create scientific illustrations and figures.
Designing Science: Adobe Illustrator for Scientists

A YouTube playlist that focuses on using Illustrator to create scientific diagrams and figures. It’s a great way to learn visually.
Coursera and LinkedIn Learning Courses

There are specific courses available on these platforms that are geared towards using Adobe Illustrator for scientific communication.

### BioRender
BioRender is more intuitive and specifically designed for scientists, so it's easier to pick up even without prior design experience. Here’s how you can get started:

BioRender Learning Hub

BioRender’s official learning platform offers tutorials, webinars, and a wide range of resources to help you get started. It’s tailored to various scientific fields, so you can find content relevant to your area.
BioRender YouTube Channel

The channel provides tutorials, tips, and webinars on using BioRender effectively. It’s especially useful for visual learners.
BioRender Templates

Explore the templates library, which includes a wide variety of pre-designed figures and diagrams. You can customize these templates to fit your research needs.
BioRender Blog

The blog offers tips and tricks on creating better scientific figures, along with updates on new features and user stories.
These resources should help you get a solid start on using both Adobe Illustrator and BioRender to create professional, publication-ready figures.
